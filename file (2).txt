# Магазин за техника - Python проект

## Описание
Проект за управление на магазин за техника.
Позволява преглед, сортиране, продажба, зареждане и намаление на продукти.

## Файлове
- models.py   : Клас Product със методи за управление
- main.py     : Главен файл - създава обекти и изпълнява операции
- requirements.txt : Използвани библиотеки (само вградени)
- readme.txt  : Описание на проекта

## Как се стартира
python main.py

## Библиотеки
Не се използват външни библиотеки.
Изисква Python 3.x
